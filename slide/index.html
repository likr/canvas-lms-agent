エージェントの賢さはどう作る？

Model Context Protocol (MCP)

  • 共通の規格！: AIがツールやデータに触るための統一的なルールです。
  • API情報を自動公開: 必要なエンドポイントやパラメータ構成をAIにダイレクトに伝達。
  • 接続情報を隠蔽: トークンなどを隠したまま、安全な実行環境で動かせます。

Agent Skills

  • 指示書をパッケージ化: ガイドライン（SKILL.md）やスクリプトを同梱。
  • 勝手にインポート: フォルダを検知して、AIが手順を自動でロードします。
  • 試行錯誤を撲滅: ハマりやすい手順を事前に叩き込んで、AIの無駄なエラーをカット！
